# Leave Management System

## 🚀 Quick Start

1. Extract the zip file
2. Open terminal in the project folder
3. Run: `npm install`
4. Run: `npm run dev`
5. Open: http://localhost:3000

## 📋 Features

### Manager Portal
1. **Add Users** - Create employee or manager accounts
2. **View Leave Requests** - Filter by department, approve/reject leaves
3. **View Details** - See complete information about each request

### Employee Portal
1. **Apply for Leave** - Submit leave requests with automatic duration calculation
2. **View My Leaves** - See all your leave requests and their status

## 🎯 Workflow

1. Manager logs in and adds users (employees/managers)
2. Employee logs in and applies for leave
3. Manager views all leave requests, filters by department
4. Manager clicks "View" to see full details
5. Manager approves or rejects with comments

## 📦 What's Included

- ✅ Login with role-based routing
